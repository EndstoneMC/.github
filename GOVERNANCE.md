# Governance

## Project Lead

Endstone is led by **Vincent Wu** ([@wu-vincent](https://github.com/wu-vincent)), who serves
as Benevolent Dictator for Life (BDFL). The BDFL has final authority on all technical and
project decisions, including:

- Architecture and API design
- Release schedules
- Repository and team management
- Licensing decisions
- Accepting or rejecting contributions

## Decision Making

Day-to-day decisions (bug fixes, minor features, documentation) are made through the normal
pull request process. Any maintainer with write access to a repository can review and merge
PRs that align with the project direction.

For significant changes (new APIs, breaking changes, new repositories, licensing), the BDFL
makes the final call. Community input is welcome and encouraged via GitHub Issues and Discord.
